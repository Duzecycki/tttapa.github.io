Detailed Description

A class for MIDI_Control_Elements that read from a digital input and send either MIDI Note events or MIDI Control Change events.

The input is debounced, the debounce time can be changed in Settings.h.

Examples:
Ex.02.Button.ino, Ex.06.Banks.ino, Ex.07.BankSelectors.ino, Ex.09.Bank-Calibrate.ino, Ex.10.Debug-MIDI.ino, Ex.14.LED.ino, Ex.17.Transposer.ino, and OLED-Demo.ino.

enum Digital::MessageType

The message type for this element.

Enumerator
NOTE 

Send MIDI Note On events when the button is pressed and Note Off events when the button is released.

CC 

Send MIDI Control Change events with a value of velocity when the button is pressed and Control Change events with a value of 0 when the button is released.

velocityThe MIDI Note velocity (messageType == NOTE) [1, 127] or Controller value when on (messageType == CC) [1, 127].
messageTypeSets the MIDI message type. Either Digital::NOTE or Digital::CC.
