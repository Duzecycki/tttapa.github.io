Detailed Description

A class of MIDIOutputElements that read the input of a latching push button or toggle switch, and send out MIDI Control Change events.

When the switch changes state, two MIDI events are sent: the first one with a value of 0x7F, followed immediately by a second one with a value of 0x00.
The switch is debounced in software.
This version cannot be banked.
