Detailed Description

template<uint8_t N>
class VolumeControl< N >

A class for controlling the volume of AudioMixer4 objects using a potentiometer.

Template Parameters
NThe number of mixers.
Examples
OLED-Demo-Audio.ino, VUMeterLEDs-USB-DAC.ino, and VUMeterOLED-USB-DAC.ino.
