Detailed Description

template<uint8_t N>
class AudioVULEDs< N >

A VU meter that reads from an Audio stream using the AudioAnalyzePeak class, and displays it on an LED bar graph.

Template Parameters
NThe number of LEDs.
Examples
VUMeterLEDs-USB-DAC.ino.
