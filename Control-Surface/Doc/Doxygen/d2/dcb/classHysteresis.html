Detailed Description

template<uint8_t BITS>
class Hysteresis< BITS >

A class for applying hysteresis to a given input. 

     This reduces the noise by decreasing the resolution, and it prevents
     flipping back and forth between two values.

An example for BITS = 7 and an input from 0 to 1023
