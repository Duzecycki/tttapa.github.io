Detailed Description

template<uint8_t N>
class MCU::Bankable::VPotRing< N >

A class for MIDI input elements that represent Mackie Control Universal V-Pots.
This version can be banked.

Template Parameters
NThe number of banks.
Examples
OLED-Demo-Audio.ino, and OLED-Demo.ino.
