Detailed Description

template<setting_t N, class Sender>
class ManyAddresses::MIDIFilteredAnalogAddressable< N, Sender >

A class for potentiometers and faders that send MIDI events and that have different addresses that can be selected.

The analog input is filtered and hysteresis is applied.

See also
FilteredAnalog
