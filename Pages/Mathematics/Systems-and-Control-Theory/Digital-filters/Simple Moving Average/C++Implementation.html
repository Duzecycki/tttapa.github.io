Simplifying the Difference Equation

Recall the Simple Moving Average difference equation: y[n]=1NN−1∑i=0x[n−i](1) A naive approach would be to implement the difference equation directly: keeping the last N−1 inputs, and calculate the sum on each iteration, calculating N−1 additions at each time step.
However, we can do much better if we notice how only two terms of the sum change each time: y[n+1]=1NN−1∑i=0x[n+1−i]=1N(x[n+1]+N−1∑i=1x[n+1−i])=1N(x[n+1]+N−1∑i=1x[n+1−i]+x[n+1−N]−x[n+1−N])=1N(x[n+1]+N∑i=1x[n+1−i]−x[n+1−N])=1N(x[n+1]+N−1∑i=0x[n−i]−x[n+1−N])=y[n]+1N(x[n+1]−x[n+1−N]) We can now define the sum S[n] as follows: S[n]≜N⋅y[n]=N−1∑i=0x[n−i]⇔y[n]=S[n]/N The difference equation then becomes: S[n+1]=S[n]+x[n+1]−x[n+1−N](2) To update the sum, each iteration now requires only one addition and one subtraction, as well as some housekeeping to remember the previous inputs. To get the output y[n], a division by N is needed.

C++ Implementation

We can now implement Equation 2 directly, and we'll use a rounding division instead of truncating the quotient. Note that this rounding operation is valid for unsigned integer types only.
The previous inputs x[n−i] are kept in a circular buffer. 

#include <stdint.h>

template <uint8_t N, class input_t = uint16_t, class sum_t = uint32_t>
class SMA {
  public:
    input_t operator()(input_t input) {
        sum -= previousInputs[index];
        sum += input;
        previousInputs[index] = input;
        if (++index == N)
            index = 0;
        return (sum + (N / 2)) / N;
        static_assert(
            sum_t(0) < sum_t(-1),  // Check that `sum_t` is an unsigned type
            "Error: sum data type should be an unsigned integer, otherwise, "
            "the rounding operation in the return statement is invalid.");
    }

  private:
    uint8_t index             = 0;
    input_t previousInputs[N] = {};
    sum_t sum                 = 0;
};
