Get the state change, given the previous estimated state, the current sensor reading, and the current control input.

Calculates \( \hat{x}_{k+1} = \left(A \hat{x}_k + B u_k\right) \oplus L \left(y_k \ominus C \hat{x}_k\right) \)

The subtraction and addition are implemented as quaternion multiplications.

Parameters
x_hatThe previous estimated state.
y_sensorThe current sensor reading.
uThe current control input.
