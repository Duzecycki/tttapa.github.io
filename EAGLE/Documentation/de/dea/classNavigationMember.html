Detailed Description

Member of the "navigation" population used to determine the LQR cost matrices for the navigation controller. Chromosome contains 3 values for the cost matrix Q and 1 value for the cost matrix R. These values are q12, xy, vxy, q12ref. We assume symmetry in the x- and y-direction, so Q = diag([q12,q12, xy,xy,vxy,vxy]) and R = diag([q12ref,q12ref]).
