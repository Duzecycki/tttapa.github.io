Detailed Description

Member of the "attitude" population used to determine the LQR cost matrices for the attitude controller. Chromosome contains 6 values for the cost matrix Q and 2 values for the cost matrix R. These values are q12, q3, w12, w3, n12, n3, u12, u3. We assume symmetry in the x- and y-direction, so Q = diag([q12, q12,q3,w12,w12,w3,n12,n12,n3]) and R = diag([u12,u12,u3]).
