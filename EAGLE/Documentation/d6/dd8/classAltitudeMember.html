Detailed Description

Member of the "altitude" population used to determine the LQR cost matrices for the altitude controller. Chromosome contains 3 values for the cost matrix Q and 1 value for the cost matrix R. These values are nt, vz, z, ut. Thus Q = diag([nt,vz,z]) and R = ut.
