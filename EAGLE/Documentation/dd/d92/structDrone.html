Drone::VecX_t Drone::operator() ( const VecX_t &  x,
const VecU_t &  u 
)

Calculate the derivative of the state vector, given the current state x and the current input u.

\( \boldsymbol{\dot{q}} = \frac{1}{2} \boldsymbol{q} \otimes \begin{pmatrix} 0 \\ \vec{\omega} \end{pmatrix} \)

\( \dot{\vec{\omega}} = \Gamma_n \vec{n} + \Gamma_u \vec{u} - I^{-1} \left(\vec{\omega} \times I \vec{\omega} \right) \)

\( \dot{\vec{n}} = k_2 \left(k_1 \vec{u} - \vec{n}\right) \)

\( \dot{\vec{v}} = \boldsymbol{q} \star \begin{pmatrix} 0 \\ 0 \\ 1 \end{pmatrix} \frac{F_{z'}}{m} - \begin{pmatrix} 0 \\ 0 \\ g \end{pmatrix} \quad \) where \( \star \) is the quaternion rotation of a vector and \( F_{z'} = C_t \rho D_p^4 n_t^2 N_m \) is the thrust along the local z-axis

\( \dot{\vec{x}} = \vec{v} \)

\( \dot{n_t} = k_2 \left(k_1 u_t - n_t\right) \)

Parameters
xThe current state of the drone.
uThe current control input u.
Returns
The derivative of the state, x_dot.

observerThe discrete observer that estimates the system state, given the control signal u and a noisy measurement of the system output y.
