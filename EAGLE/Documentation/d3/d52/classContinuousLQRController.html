const Matrix< Ny, Nu > &  D,
bool  continuous

inlinestaticinherited

Solves the system of equations

\( \begin{cases} \dot{x} = A x + B u = 0 \\ y = C x + D u = r \end{cases} \\ \begin{cases} x_{k+1} = A x_k + B u = x_k \\ y = C x_k + D u = r \end{cases} \)
for the continuous and the discrete case respectively, for any given reference output \( r \).

Returns
A matrix \( G \) such that \( \begin{pmatrix} x^e \\ u^e \end{pmatrix} = G r \).
